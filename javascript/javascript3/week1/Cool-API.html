For purpose of this exercise I am analysing Movebank REST API.

How it works?

  • It's open source. So it's free to use.
  • One can access its database through HTTP/CSV requests or JSON/JavaScript requests.
  • Movebank provides both private and public data. To be able to use private data one needs to get permission from data provider.
  • Documentation to a call to public service can be seen here.

What kind of JSON data API responds with?

  • Data structure shows objects nested in an array.
  • Values are numbers, strings and an array of objects.
  • One can access object that holds latitude and longitude location of an animal being tracked and timestamp in UNIX time unit.
